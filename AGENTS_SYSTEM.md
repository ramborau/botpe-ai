# Agents
# BotPe AI - AI Agents System

## 5 Core Agents for MVP

### 1. Bot Builder Agent
Generates complete bot flows from natural language descriptions.

### 2. Intent Recognition Agent
Classifies user intents and extracts entities from WhatsApp messages.

### 3. Response Generation Agent
Creates natural, contextual responses with proper tone and personalization.

### 4. Template Generator Agent
Creates WhatsApp-compliant message templates following policies.

### 5. Customer Support Agent
Provides AI-powered assistance in inbox with suggested responses.
